string.c

#include <string.h>

In this context, s usually represent the pointer to the string.

  • strlen(s) → return lenght of the string

  • strcmp(s1, s2) → compare the two strings. Return 0 if the two strings are equal

  • strcpy(s1, s2) → copy s2 into s1
