Basic Concepts

Homebrew is a package manager for Mac.

  • Use xcode-select --install to install command line tools on mac

  • Copy/paste the first instruction in the official documentation to install

  • Test if the installation was successful using brew help

There are two kinds of packages: formulae usually deal with command line softwares, while casks enamble installation of mac application directly from homebrew.

Homebrew can be unsinstalled preatty simply with the command provided in the FAQ section of the website.

Basic Commands

  • brew search → lists all the packages that can be installed

  • brew search something → return packages names which contains something in the name

  • brew list → lists installed packages

  • brew install package_name → intalls the package

  • brew uninstall package_name → uninstalls the package

  • brew info package_name → return information about the package (both installed and not installed ones)

  • brew update → updates the list of available packages and their version (it does not install anything)

  • brew outdated → lists all the installed packages that have an updated version

  • brew upgrade → upgrades all the outdated packages

  • brew cleanup → delete old versions of installed packages (when packages are updated, homebrew does not replace the version but just install the new one)

Adding 3rd Parties Brew Repositories

Sometimes something is not available in the main homebrew repository. Other repositories in which the brew command can search can be added. The name of this repositories is provided in documentation of the software one wants to install.

  • brew tap repository_name → add the reposistory to homebrew. Now one can search the packages contained in that using brew search and install as usual

Using Cask

Basically just add caks after brew in each command

  • brew cask install package_name → install the cask package

  • brew cask home package_name → open the home webpage of the package
