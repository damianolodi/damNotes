Functions

All functions are reported in the documentation.

  • {{ .Site.Params.homeText | markdownify }} → access the site parameter homeText (in the config file) and apply the markdownify function

  • {{ $css := "css/bootstrap.min.css" | absURL }} → define a variable and apply the absURL function

  • {{ hugo.Generator }} → add in the html head to display the hufo version as a metadata in the source code

String Manipulation

  • {{ print SOMETHING }} → print the argument using the Golang print function. See printf to apply string format
