Accessing Hardware Info

  • lsblk → access all the information about external drives (it menas ‘list block devices’).

    • in Linux, all external storage devices are stored in /dev/sd*. Usualy, first letter (a,…) is the name of the disk, followed by a number if more that one partitions are present

  • lsusb → lists all USB peripherals (keyboard, mouse, serial ports,…)

  • lspci → lists all peripherals attached to the PCI bus

  • cat /proc/cpuinfo → contains all info on the mounted CPU

    • top → display statistics on the CPU usage

  • cat /proc/meminfo → contains all the info on the installed RAM

    • free -m → display amount of free and occupied memory

Network Cards

  • ip addr (or ifconfig, but it is deprecated) → lists all network cards

    • firts diplayed is lo (loopback). It is a virtual network card used to assign the ip address for localhost

    • eth0 (on Debian) is the cabled network card. inet and inet6 are the ip address of the local network

  • ip route → list the ip address of the router used to connect to external networks

  • host website_name → return ip address and hosting informations about the provided website

  • ip link set interface_name down → turn off the ntetwork card specified in interface_name (e.g. eth0). Use up to reactivate it

  • ip address change new_ip_address dev interface_name → change the ip address of the specified interface. This modification is temporary (until next reboot of the machine)

    • the ip address can be modified permanently changing the configuration inside a directory in /etc/ (the specific file depends on the distro)
