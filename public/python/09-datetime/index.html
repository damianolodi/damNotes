Datetime

datetime Submodule

  • dt_obj = dt.datetime(yyyy, mm, dd, hh, mm, ss, ususus) → create datetime object. Don’t use leading zeros

    • dt_obj.date(), dt_obj.time(), dt_obj.year, ... → acess onyl part of the object

  • dt.datetime.today() → return current local date and time with a timezone specified as None

  • dt.datetime.now() → return current local date and time in standard format. It has a parameter to specify the time zone

    • dt.datetime.now().time() → return only the date

  • dt_obj.strftime('%Y-%m-%d %H:%M:%S.%f') → format the datetime object and return a string

  • dt.datetime.strptime('date time', 'format') → convert a string into a datetime object following the provided format

date Submodule

  • dt.date(yyyy, mm, dd) → create date. Don’t use leading zeros

  • tday = dt.date.today() → get current global date

    • tday.year, tday.month, tday.day → get only part of the date

    • tday.isoweekday() → get day of the week (monday is 1, sunday is 7)

    • tday.weekday() → get day of the week (monday is 0, sunday is 6)

time Submodule

  • t = dt.time(hh, mm, ss, ususus) → create a time object

    • t.hour, t.minutes, t.seconds, ... → acess only part of the object

timedelta Submodule

Timedeltas objects can be added/subtracted to dates, or they can be obtained subtracting two dates/times/datetimes.

  • td = dt.timedelta(days=7) → create a timedelta object long 7 days

  • td.days → return only the “days” of the timedelta object

  • td.total_seconds() → convert timedelta object in seconds

pytz module

import datetime as dt
import pytz

It is recommended to use this when working with timezones

  • dt_obj = dt.datetime(yyyy, mm, dd, hh, mm, ss, tzinfo=pytz.UTC) → create a timezone-aware datetime object

  • dt_utc = dt.datetime.now(tz=pytz.UTC) → get current timezone-aware date and time

  • dt_utc.astimezione(pytz.timezone('timezone/name')) → convert to a different timezone. To see al timezone names see the doc or print the pytz.all_timezones list

  • convert “naive” datetimes into timezone-aware ones using

    my_dt = dt.datetime.now() # 'naive' datetime object
my_timezone = pytz.timezone('timezone/name')
dt_with_timezone = my_timezone.localize(my_dt)

dateutil module

This module expands the datetime module.

  • To check if a string represent a date use

    from dateutil.parser import parse

try:
    parse(string, fuzzy=False)
except ValueError:              # it is not a date
    pass

Time

  • time.perf_counter() → return value (in seconds) of a performance counter (calcualted from the highest clock). Used to measure performance. The single value is useless, it is required to make difference between consecutive results (doc)
