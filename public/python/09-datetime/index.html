import datetime as dt

  • dt.datetime.now() → return current date and time in standard format

    • dt.datetime.now().strftime('%Y-%m-%d %H:%M:%S.%f') → format the returned valued

    • dt.datetime.now().time() → return only the date

  • datetime.today() → return current date
